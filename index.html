CONTROL MATRIX

Complete control over your network topology and player movement.

Command Permission Node
/nexus join <server> autonexus.command.join (configurable)
/nexus servers autonexus.admin

# config.yml
server-name: "auto"
# auto = detect from server.properties or folder name
# or set fixed id manually, e.g.:
# server-name: "survival-1"
group: "auto"
# auto = example group name; set your own like "lobby"
redis:
  host: "127.0.0.1"
  port: 6379
network:
  namespace: "global"
  heartbeat-interval: 5
  cleanup-threshold: 15
commands:
  join:
    require-permission: false
# false = /nexus join for everyone; true = needs 'autonexus.command.join'
# That's it. No complex SQL.
